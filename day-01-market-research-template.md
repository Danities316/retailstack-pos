## рҹ“Ӣ **Step-by-Step Creation in Canva**

### **Step 1: Setup**
1. Open Canva.com
2. Search for "Twitter Post" template
3. Choose 1200x675px size

### **Step 2: Background**
1. Add gradient background
2. Use dark blue to light blue
3. Set opacity to 80%

### **Step 3: Header**
1. Add text box for title
2. Use bold white font
3. Center align

### **Step 4: Main Content**
1. Create 3 equal columns
2. Add icons from Canva library
3. Add percentage circles
4. Add descriptive text

### **Step 5: Styling**
1. Use consistent colors
2. Add subtle shadows
3. Ensure good contrast
4. Test readability

### **Step 6: Export**
1. Download as PNG
2. High quality (300 DPI)
3. Save as "day-01-market-research.png"
